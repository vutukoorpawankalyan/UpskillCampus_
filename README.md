# Smart Home Automation using IoT

## Overview
This project is a Smart Home Automation system that allows users to control and monitor home appliances using IoT technology. It enables remote operation of devices through a Wi-Fi connection.

## Features
- Control lights and fans remotely
- Real-time device monitoring
- Wi-Fi-based communication
- Energy-efficient automation
- Easy to expand with additional sensors

## Components Used
- ESP8266/ESP32 (or Arduino Uno)
- Relay Module
- Wi-Fi Module (if using Arduino)
- LEDs/Bulbs
- Power Supply
- Jumper Wires

## Software Used
- Arduino IDE
- Blynk / MQTT (Optional)

## How to Run
1. Open the code in Arduino IDE.
2. Install the required libraries.
3. Update the Wi-Fi credentials.
4. Upload the code to the ESP8266/ESP32 or Arduino.
5. Control the devices using the connected application.
