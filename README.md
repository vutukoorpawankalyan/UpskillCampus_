# 🏠 House with Earthquake Warning Alarm using IoT

## 📌 Project Overview
The **House with Earthquake Warning Alarm using IoT** is an intelligent safety system that detects earthquake vibrations and immediately alerts occupants through a buzzer and warning indicators. The system uses IoT technology to monitor seismic activity in real time and can send notifications to users, helping improve safety during earthquakes.

---

## 🚀 Features
- 🌍 Earthquake vibration detection
- 🚨 Instant buzzer alarm
- 💡 LED warning indicator
- 📱 IoT-based remote monitoring
- 📊 Real-time sensor data
- ⚡ Fast emergency alerts
- 🔋 Low power consumption
- 🏡 Suitable for homes and small buildings

---

## 🛠️ Components Required
- Arduino UNO / ESP8266 NodeMCU
- Vibration Sensor (SW-420 or similar)
- Buzzer
- LED
- Breadboard
- Jumper Wires
- USB Cable
- Power Supply

---

## ⚙️ Working Principle
1. The vibration sensor continuously monitors ground vibrations.
2. If vibrations exceed the predefined threshold, the controller detects a possible earthquake.
3. The buzzer sounds immediately to alert occupants.
4. The LED flashes as a visual warning.
5. The IoT module uploads the alert and sensor data to a cloud platform.
6. Users can monitor alerts remotely through a mobile app or web dashboard.
